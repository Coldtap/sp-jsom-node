# Installation

## Prerequisites

### Node.js 6 or higher

### Global NPM modules

```bash
npm install typescript ts-lint ts-node -g
```

## Restore dependencies

```bash
npm install
```

## Run (.ts directly)

```bash
npm run exec:ts
```

or

```bash
ts-node ./src/index
```

## Build

```bash
npm run build
```

## Run (transpiled .js)

```bash
npm run exec:js
```
